**文字列の各単語**の最初の文字を大文字に変換し、残りをそのままにする `CapitalizeWords <T>` を実装します。

例えば

```ts
type capitalized = CapitalizeWords<'hello world, my friends'> // expected to be 'Hello World, My Friends'
```
